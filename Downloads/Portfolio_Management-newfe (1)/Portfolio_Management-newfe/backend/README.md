# Portfolio Management Project
Spring Boot backend

## Steps to set up the db
1. **Pull or clone the repository**
   ```bash
   git clone <repo-url>  (one time only)
   git pull
2. **Create an empty MySQL database (ONE TIME ONLY)**
   ```bash
   CREATE DATABASE portfolio_db;
3. **Run the PortfolioManagerApplication**: To create tables and feed initial data
4.  **All mock/demo data is centrally managed in `data.sql` located at `src/main/resources/`**
